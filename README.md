# La Proporción Áurea y la Sucesión de Fibonacci

Este es un proyecto web educativo e interactivo diseñado para explorar en profundidad el fascinante mundo del **Número Áureo (Φ)** y su estrecha relación con la **Sucesión de Fibonacci**.

## 📖 Descripción del Proyecto
A través de diferentes secciones, la página web explica de forma visual e intuitiva cómo estas proporciones matemáticas perfectas se manifiestan en diferentes disciplinas a lo largo de la historia y en nuestro entorno. 

El proyecto se divide en las siguientes páginas temáticas:
- **Inicio:** Introducción general a la Proporción Áurea.
- **El Número Áureo:** Explicación matemática, la serie de Fibonacci y la Espiral Logarítmica.
- **Arte y Arquitectura:** La influencia del número Phi desde el Partenón hasta el Renacimiento (Da Vinci, Luca Pacioli).
- **En la Naturaleza:** Patrones áureos en girasoles, moluscos y galaxias.
- **La Divina Proporción:** El aspecto estético y filosófico en el Renacimiento.
- **Un Patrón de la Naturaleza:** Reflexión sobre la funcionalidad biológica de estos patrones.
- **Actividades:** Sección interactiva para poner a prueba los conocimientos adquiridos.

## 🚀 Características Interactivas
La sección de **Actividades** incluye herramientas desarrolladas en JavaScript puro:
1. **Generador de Fibonacci:** Calcula el número correspondiente a cualquier posición de la secuencia.
2. **Generador de la Espiral de Durero:** Utiliza HTML5 `<canvas>` para dibujar en tiempo real la espiral áurea según el número de iteraciones indicado.
3. **Cuestionario de Conocimientos:** Un *quiz* evaluable de 20 preguntas (opción múltiple, verdadero/falso y texto libre) sobre historia, matemáticas y geometría.
4. **Material Audiovisual:** Integración de documentales y vídeos explicativos incrustados de forma *responsive*.

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructura semántica.
- **CSS3:** Diseño responsivo (adaptable a móviles y escritorio), animaciones suaves, variables de color y tipografía de Google Fonts (Inter).
- **JavaScript (ES6):** Lógica del cuestionario, cálculos matemáticos y renderizado en Canvas.

## ⚙️ Instalación y Uso
Al ser un proyecto estático, no requiere de una base de datos ni procesos de compilación complejos.

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador web preferido.

*Nota sobre los vídeos integrados:* Si abres los archivos haciendo doble clic directamente (con la ruta `file:///...`), es posible que los reproductores como YouTube muestren un error por políticas de privacidad del navegador. Para disfrutar de toda la funcionalidad, se recomienda abrir el proyecto utilizando un servidor local (por ejemplo, *Live Server* en VSCode o ejecutarlo a través de un servidor como *Tomcat* en Eclipse).
